电商项目运行环境配置步骤

1. 项目初始化步骤

  1. 安装 vue 脚手架
  2. 通过 vue 脚手架创建项目
    1. 打开 cmd 命令行，输入 vue ui，打开 vue 可视化面板
  3. 配置 vue 路由
  4. 配置 element-UI 组件库
  5. 配置 axios 库
  6. 初始化 git 远程仓库
  7. 将本地项目托管到 GitHub

后台项目的环境安装配置

  1. 安装 MySQL 数据库
  2. 安装 node.js 环境
  3. 配置项目相关的信息
  4. 启动项目
    1. 后台数据文件 vue_api_server，先 npm install 安装依赖项，之后输入 node .\app.js
    2. 如果没有报错，就证明正常
  5. 使用 postman 测试后台项目接口是否正常
