1. git

  • 最常用的代码版本管理用具
  • 大型项目需要多人协作开发，必须熟用 git
  • Mac OS 自带 git 命令，windows 可以去官网下载安装
  • git 服务端常见的有 GitHub coding.net 等
  • 大公司会搭建自己的内网 git 服务

1.1 常用的 git 命令

  • git add . 提交所有的文件
  • git checkout 文件名 撤销某个文件内容的修改
  • git commit -m “xxx” 添加一条提交记录
  • git push origin master 提交 master 分支代码
  • git pull origin master 拉取 master 分支代码
  • git branch 查看当前所在的分支
  • git checkout -b xxx / git checkout xxx 切换到其他分支
  • git merge xxx 把 xxx 分支的代码合并到 master 分支
  • git stash 把修改暂存起来
  • git stash pop 把暂存的修改拉回来

5. Linux 常用命令

  • 公司的线上机器一般都是 Linux （参考阿里云）
  • 测试机也需要保持一致，用 Linux
  • 测试机或者线上机出了问题，本地又不能复现，需要去排查
  • ls 查看文件
  • ls-a 查看所有文件
  • ll 查看文件列表
  • mkdir xxx 创建文件夹
  • ll xxx 查看文件夹
  • rm -rf xxx 删除文件夹及其所有内容
  • cd xxx 切换到 xxx 目录
  • mv xxx.html xxx1.html 修改文件名
  • mv xxx.html ../xxx.html 移动文件到上级目录
  • cp xxx.html xxx.html 拷贝文件
  • rm xxx.html 删除文件
  • touch xxx.html 新建文件
  • vi xxx.html 新建并打开文件
    • 按 i 键进入编辑
    • 按 ESC 键退出编辑区域
    • 按 shift + ：键，并输入 w，保存输入的内容
    • 按 shift + ：键，并输入 q，退出vim编辑
    • 按 shift + ：键，并输入 q!，不保存内容强制退出vim编辑
  • vim xxx.html 查看文件内容
  • cat xxx.html 打印出文件的所有内容，在控制台查看
  • head xxx.html 打印出前几行，在控制台查看
  • tail xxx.html 打印出末尾几行，在控制台查看
  • grep “aaa” xxx.html 在 xxx 文件中查找 aaa 关键字的内容
